Initio Display Manager

Initio — это минималистичный оконный менеджер для X11, полностью написанный на Python. Проект предоставляет базовый экран входа в систему (GUI) и простую среду рабочего стола с панелью задач и управлением окнами.

🚀 Имеет:

Экран входа (DM): Аутентификация через PAM (Pluggable Authentication Modules).

Оконный менеджер (WM):

Интеграция с rofi для запуска приложений.

Автоматическая настройка: Скрипты для установки системных зависимостей и регистрации сессии в X11.

📦 Требования

ОС: Linux (поддерживаются системы с pacman или apt).

Python 3.11+.

X11 Server.

🛠 Установка
1. Клонирование
git clone https://github.com/QLegacy/initio.git
cd initio
2. Подготовка окружения
При первом запуске main.py автоматически вызывает установщик зависимостей (libs.py), который:
Установит системные пакеты (python-pam, python-xlib, rofi, tkinter и др.).
Установит необходимые Python-библиотеки.
Создаст .desktop файл в /usr/share/xsessions/ для выбора Initio в меню вашего экрана входа.
3. Сборка (опционально)
Если вы хотите скомпилировать проект в бинарный файл:
python3 build.py
Скрипт использует PyInstaller и выведет детальные инструкции по установке скомпилированного файла в /opt и настройке прав (SUDO/SUID).
🖥 Использование
После установки вы можете выйти из текущей графической сессии и выбрать "Initio" в меню выбора сессии вашего дисплейного менеджера (например, SDDM, GDM, LightDM).
Вход: Введите имя пользователя и пароль.
Запуск приложений: Нажмите синюю кнопку на панели в левом нижнем углу — откроется rofi.
Управление окнами:
Перетаскивание: ЛКМ по заголовку окна.
Изменение размера: ЛКМ по краям окна.
⚠️ Важное предупреждение
Права доступа: Для работы PAM-аутентификации бинарному файлу могут потребоваться повышенные привилегии (SUID бит). Будьте осторожны при настройке прав доступа.
Зависимости: Скрипт libs.py использует флаг --break-system-packages, так как он устанавливает Python-пакеты глобально для корректной работы с системным интерпретатором.
